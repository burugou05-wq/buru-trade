# THE WALL STREET - 株取引シミュレーター

本格的なブラウザベースの株取引ゲーム。リアルタイムで変動する株価に対応しながら、資産を増やして投資目標を達成するシミュレーター。

## 🎮 ゲーム概要

### 目的
初期資金から資産を増やし、以下のミッションをクリアすること。

| ミッション | 報酬 |
|----------|------|
| 資産100万円達成 | 1000点 |
| 資産1000万円達成 | 5000点 |
| ダブルゲイン(1銘柄で2倍化) | 2000点 |

### 難易度

#### 🟢 イージー
- 初期資金: **500万円**
- マクロ変動: 小さい
- ニュース発生: 少ない
- 初心者向け

#### 🔵 ノーマル (推奨)
- 初期資金: **100万円**
- マクロ変動: 中程度
- ニュース発生: 標準
- バランス型

#### 🔴 ハード
- 初期資金: **100万円**
- マクロ変動: 大きい
- ニュース発生: 多い
- 上級者向け

## 📊 ゲーム画面

### ヘッダー
- **総資産額**: 現在の資産合計
- **現金残高**: 投資に使える現金
- **進行コントロール**: 再生/一時停止、速度変更
- **マクロ経済**: 現在の経済状況

### 左パネル: 市場銘柄
8つの上場企業の株価をリアルタイム表示。
- **選択済み**: 青色でハイライト
- **価格変動**: 上昇は緑、下降は赤で表示
- **変動率**: 前日比をパーセンテージで表示

### 中央パネル: チャートと取引
- **チャート**: 過去60日間の株価推移
  - 上昇トレンド: 緑色
  - 下降トレンド: 赤色
- **価格情報**: 現在価格、変動幅、PE、配当利回
- **取引フォーム**: 購入/売却数量の指定と実行

### 右パネル: ポートフォリオとミッション
- **ポートフォリオ**: 保有株式一覧
  - 銘柄ごとの価値を円グラフ的に表示
  - 含み損益をカラフルに表示
- **ミッション進捗**: 達成度をプログレスバー表示

### フッター: ニュースティッカー
最新のマーケットニュースをスクロール表示。

## 🎯 基本操作

### 1. 難易度選択
起動時に難易度を選択します。
```
イージー → ノーマル → ハード
```

### 2. 企業を選択
左のマーケットリストから企業をクリック。

### 3. チャートを分析
過去の株価推移からトレンドを判断します。

### 4. 取引を実行
```
取引数量を入力 → 購入/売却ボタンをクリック
```

**クイック入力:**
- `+10`, `+100`: 数量を増減
- `MAX`: 購入可能な最大数量

### 5. ゲーム進行
```
再生ボタンをクリック → 時間が自動で経過
```

**速度調整:**
- `x1`: 標準速度
- `x2`: 2倍速
- `x3`: 3倍速

## 💡 投資戦略のコツ

### 分散投資
複数の企業に投資することでリスク軽減。

### ニュース活用
マーケットニュースが株価に大きく影響。
- 🟢 好材料 → 上昇期待
- 🔴 悪材料 → 下落警戒

### マクロ経済
全体的な経済状況:
- 📈 好景気 → 全般的に上昇
- 📉 不況 → 全般的に下降

### テクニカル分析
チャートの形状からトレンドを読む:
- 右肩上がり → 買いシグナル
- 右肩下がり → 売りシグナル

### 利確・損切り
タイミングが重要:
- 目標利益に達したら早期に利確
- 損失が膨らむ前に損切り

## 🎮 ゲームメカニクス

### 株価変動要因
1. **ランダム変動** (日々の投機的売買)
2. **業績サイクル** (企業固有のサイクル)
3. **マクロ経済** (全体的な景気変動)
4. **ニュースイベント** (突発的な大きな変動)

### マッチング
各企業に以下の属性があります:
- **PE (株価収益率)**: 企業の割高/割安の目安
- **配当利回**: 保有による利益

## 🚀 クイックスタート

### 初めての方
1. ゲームを起動
2. チュートリアルを読む
3. **ノーマル難易度** を選択
4. まずは1-2社に少額投資してみる
5. 株価の動きを観察

### ハードプレイを目指す方
1. **ハード難易度** でスタート
2. 複数企業の相関関係を分析
3. ニュースを活用したスイングトレード
4. 資産1000万円達成を目指す
